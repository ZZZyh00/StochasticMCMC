######################################################################

Gaussian toy example

######################################################################

This is a single mode Gaussian model example for posterior sampling. We generate the observed samples from a gaussian whose mean is randomly chosen from [0, 1]. The covariance is taken as I. 

Once these observations are generated, we would like to impose a bayesian prior for our parameter, mu (the mean). We take the same covariance for posterior i.e. I. For this posterior sampling of mu, we run HMC. 

generatePlot shows the grouth truth posterior that can be calculated analytically (which is Gaussian again) and the samples generated by HMC.
